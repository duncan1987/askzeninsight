# 出海网站合规文档通用模板

**文档版本：** v1.0
**适用范围：** 面向全球用户的SaaS/互联网产品

## 第一部分：隐私政策（Privacy Policy）

### 1.1 政策引言

**模板条款：**

```
[网站名称]（"我们"、"我们公司"、"我们"）尊重您的隐私，并致力于保护您通过我们的服务[网站URL]（"服务"）提供的个人信息。本隐私政策解释了我们如何收集、使用、披露和保护您的信息。使用我们的服务即表示您同意本隐私政策中描述的做法。

最后更新日期：[日期]
生效日期：[日期]
```

**关键要点：**
- ✅ 明确主体身份（公司名称）
- ✅ 覆盖范围（网站/服务）
- ✅ 更新日期（GDPR要求）
- ✅ 生效日期
- ✅ 同意使用即接受

**模板条款：**

```
我们收集您在使用服务时主动提供给我们的信息：

**账户信息**
- 姓名、电子邮件地址

**支付信息**
- 信用卡/借记卡信息
- 账单地址
- 交易历史

**服务使用信息**
- 您创建的内容（消息、文件等）

- 偏好设置（语言、通知等）

**通信信息**
- 您与我们沟通时提供的信息
- 支持请求和反馈
```

**关键要点：**
- ✅ 分类清晰（账户/支付/使用/通信）
- ✅ 示例具体
- ✅ 涵盖所有收集场景

#### 1.2.2 自动收集的信息

**模板条款：**

```
当您访问或使用我们的服务时，我们自动收集某些信息：

**设备和连接信息**

**使用信息**

**Cookie和类似技术**

- 认证令牌

```

**GDPR要求：**
- ✅ 说明法律依据（合法利益、合同履行等）
- ✅ Cookie同意管理
- ✅ 位置信息需特殊说明

### 1.4 信息共享与披露

**模板条款：**

```
我们不会出售、出租或交易您的个人信息。我们可能在以下情况下共享信息：

**服务提供商**
- 支付处理商（如Stripe、PayPal、Creem）
- 云服务提供商（如AWS、Vercel、Supabase）
- 分析服务（如Google Analytics、Mixpanel）
- 邮件服务（如SendGrid、Mailgun）

这些第三方仅按我们的指示访问您的信息，并受严格的保密义务约束。

**业务转让**
- 如果我们涉及合并、收购或资产出售，您的信息可能被转让
- 我们会提前通知您，并提供选择退出的权利

**法律要求**
- 遵守法律义务、法院命令或传票
- 保护我们的权利、财产或安全
- 防止欺诈、滥用或非法活动
- 在适用法律要求或允许的范围内

**经您同意**
- 经您明确同意的其他共享场景
```

**关键要点：**
- ✅ 明确不出售（重要！）
- ✅ 列举服务提供商类别
- ✅ 强调保密义务
- ✅ 业务转让的通知义务
- ✅ 法律要求的披露

**GDPR要求：**

### 1.6 您的权利

**模板条款：**

```
根据适用法律（如GDPR、CCPA），您享有以下权利：

**访问权**
- 请求查看我们收集的您的个人信息副本

**更正权**
- 更新不准确或不完整的信息
- 根据服务使用自动更正

**删除权**
- 请求删除您的个人信息
- 例外情况：法律要求、欺诈检测、技术限制

**限制处理权**
- 要求限制某些处理活动
- 例外：法律要求、我们或第三方的合法利益

**数据可携带权**
- 以结构化、常用格式接收您的信息
- 在技术可行的情况下传输给其他服务

**反对权**
- 反对基于合法利益的处理
- 任何时候选择退出营销

**撤回同意**
- 任何时间撤回之前给予的同意
- 撤回不影响撤回前的处理合法性

**CCPA权利（加州居民）**
- 知悉权、删除权、选择退出权
- 不歧视权（不会因行使权利而受到不同待遇）

**如何行使权利**
- 通过[邮箱]联系我们
- 通常在[X]天内响应
- 某些请求可能需要身份验证
```

**关键要点：**
- ✅ 列举GDPR所有权利
- ✅ 包含CCPA特殊权利
- ✅ 行使方式清晰
- ✅ 响应时间承诺
- ✅ 身份验证说明

```
- 终止该账户

```

**COPPA要求（美国）：**
- ✅ 明确年龄限制

### 3.3 退款计算方法

**模板条款：**

```
**计算公式**
退款金额 = 订阅价格 × (1 - 已使用比例)

**使用比例计算**

示例1：月付订阅
- 计划价格：$9.99
- 计划天数：30天
- 使用天数：3天
- 使用比例：3/30 = 10%
- 退款金额：$9.99 × (1 - 10%) = $8.99

示例2：按使用量
- 每日配额：100次操作
- 已使用：20次操作
- 使用比例：20/100 = 20%
- 退款金额：$9.99 × (1 - 20%) = $7.99

**最低退款**
如果计算退款低于$1.00，不予退款
```

**关键要点：**
- ✅ 公式透明公开
- ✅ 示例清晰
- ✅ 最低金额说明
- ✅ 多种计算方式
